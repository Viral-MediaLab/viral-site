Media Matrix

We present a scalable guide to an almost infinite array of visual media including published video, stored material, "channels" created by your friends, and web material. The material is organized as a 3-D graphical grid, organied by track and publication time. It this builds on the familiar format of a television program guide, with the additional ability to peruse any media format, the inclusion of social data, links between content, and direct access to all material. There are no modes and no divergent access styles. It is a also a scratchpad and creation surface that allows viewers to coordinate the presentation and access to any media with others.
